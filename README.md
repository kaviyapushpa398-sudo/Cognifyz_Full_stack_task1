# Cognifyz_Full_stack_task1
# 🌐 HTML Structure & Basic Server Interaction

## 📌 Overview

This project introduces the fundamentals of **server-side rendering** and **form handling** using a simple Node.js server. It demonstrates how user input from a web form is processed on the server and rendered dynamically.

---

## 🎯 Objective

To understand how frontend forms interact with a backend server and how dynamic HTML pages are generated using server-side rendering.

---

## 🏢 Project Source

This project was completed as part of a task provided by **Cognifyz**.

---

## 🛠️ Features Implemented

### ✅ 1. HTML Form Creation

* Designed a basic HTML structure
* Created forms to collect user input

---

### ✅ 2. Node.js Server Setup

* Built a simple server using Express.js
* Handled incoming HTTP requests

---

### ✅ 3. Form Submission Handling

* Created server-side endpoints to process form data
* Used POST requests to send data from client to server

---

### ✅ 4. Server-Side Rendering (EJS)

* Used EJS templates to generate dynamic HTML
* Displayed user input on the webpage after submission
